# Système de reconnaissance Faciale
Problématique : comment concevoir un système informatique fiable, permettant d’identifier automatiquement les étudiants et de gérer leurs présences en temps réel grâce à la reconnaissance faciale ? 

Objectif : développer un système de gestion automatique des présences étudiantes fondé sur la reconnaissance faciale. Il s’agira notamment de :
•	Concevoir l’architecture du système à l’aide de diagramme UML ;
•	Identifier les fonctionnalités principales et les interactions entre les acteurs ;
•	Mettre en œuvre un module de reconnaissance faciale s’appuyant sur des modèles deep learning ;
•	Générer des rapports d’assiduité exploitables par l’administration.
