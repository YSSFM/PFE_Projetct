📌 Présentation de l'application

RecoJY (Jean et Youssouf)

est un système innovant de reconnaissance faciale conçu pour automatiser la gestion des présences des étudiants. Développé dans le cadre d'un Projet de Fin d'Études (PFE), cette application utilise des algorithmes d'intelligence artificielle (DL et CNN) pour identifier les visages en temps réel et marquer automatiquement les présences.

⚙️ Fonctionnalités principales

  • Reconnaissance faciale en temps réel via webcam
  • Enregistrement automatique des présences
  • Gestion complète des étudiants (CRUD)
  • Génération de rapports PDF
  • Interface administrateur sécurisée
  • Base de données MySQL pour le stockage

🛠️ Technologies utilisées

  • Python / Flask - Backend
  • Face Recognition / OpenCV - Reconnaissance faciale
  • MySQL - Base de données
  • HTML5 / CSS3 - Frontend
  • ReportLab - Génération PDF

👤 À propos des développeurs

Ce projet a été développé par Youssouf Moussa et Bogou Dametote Jean, étudiants en DUT en Informatique à l'ESTM au Maroc. Passionnés par l'intelligence artificielle et les technos émergentes, ce projet vise à démontrer l'application pratique de la reconnaissance faciale dans le domaine de l'éducation.
